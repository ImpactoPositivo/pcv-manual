Gerenciando usuários

Definição da funcionalidade

A função “usuários” foi feita para o gestor dar poderes amplos ou mais limitados para determinados colaboradores usar a ferramenta Dialox, sendo sua atuação na plataforma totalmente monitorada pelos gestores com poderes de administração.

Para que serve?

Essa funcionalidade foi desenvolvida com a finalidade de dar mais dinamismo ao uso da ferramenta e permitir de gestores deleguem a colaboradores de um ou mais departamentos de sua instituição poderes para gerenciar de forma individualizada a comunicação com clientes.

UsuáriosNovo

Nesse tópico de menu o gestor consegue criar novos usuários, que, a partir dai poderão ter acesso a funcionalidade para enviar e responder mensagens SMS para um ou mais contatos registrados na plataforma.

Para criar um novo usuário basta inserir o endereço eletrônico corporativo do colaborador no campo (1), informar o telefone celular do mesmo no campo (2), no campo (3), escolher se aquele colaborador terá poderes de Administrador, ou se ele será um usuário comum.

A caixa de seleção “Ativo”, campo (4), já está por definição selecionada em Sim, a partir da isenção dos dados acima, basta clicar no botão Gravar, para o novo usuário ser criado.

OBSERVAÇÃO:

A partir da criação do novo usuário, o mesmo receberá um e-mail automático da plataforma comunicando seu novo status. O usuário poderá se logar automaticamente, desde que o mesmo seja usuário ativo do sistema prefeitura.com.vc.

Tipo de Usuário:

  1. Administrador: Permite ao usuário transferir saldos, consultar informações de uso, além de oferecer ao mesmo, poderes para o uso de todos os menus de serviço que o permitem enviar mensagens para um ou mais contatos cadastrados no sistema.
  2. Comum: Permite ao usuário criar campanhas, mensagens e utilizar todas as funcionalidades relativas a emissão de mensagens para um ou mais contatos registrados no sistema.

UsuáriosCadastrados

No tópico cadastrados, você acessa todos os contatos usuários com poderes para operar a plataforma na plataforma.

Como todas as rotinas da funcionalidade Comunicação Ativa, a tela de usuários cadastrados foi feita para trazer intuitividade nesta consulta. Assim sendo, o campo (1), ajusta o níumero de usuários listados em cada consulta. Isto é muito útil se sua instituição tem um grande número de usuários com poderes para operar essa funcionalidade.

O campo (2) permite que você busque um determinado usuário, como sempre, a partir de três caracteres registrados por você nesse campo, sejam eles numérico ou alfabético, nossa plataforma buscará o usuário, independentemente da posição em que àqueles caracteres apareçam no campo de e-mail, ou no campo de número de celular.

O campo (3) nos inddica uma linha contendo um registro padrão da consulta de usuários cadastrados. Nessa linha podemos ver que a tela de usuários cadastrados nos mostra a relação de usuários listando nessa ordem, endereço eletrônico, número de celular, e categoria do usuário.

O campo 4 nos mostra os dois botões que aparecem a direita de cada um dos usuários cadastrados, isto é, Editar, Excluir.

Clicando no botão Editar, você será conduzido para a tela abaixo, na qual você poderá editar qualquer uma das informações dos usuários. Esta opção somente está disponível para os usários administradores.

Ao clicar no botão Excluir, o usuário será totalmente excluido, independentemente do fato de ele ter ou não saldo remanescente de mensagens para enviar.

Atenção:

Apartir da exclusão de um usuário do sistema, o mesmo perde as permissões de utilização das rotinas de remessa de emnsagens SMS para contatos registrados. A exclusão de um usuário também apagará todo o histórico de utilização da funcionalidade Comunicação Ativa daquele colaborador.

UsuáriosTransferência

O sub menu Transferência, permite que o usuário Administrador credite ou debite saldos de mensagens para os demais usuários utilizarem.

Se você é um usuário de categoria "Admministrador", você poderá fazer transferências,conforme a tela mostrada acima.

Procedimento para transferências de saldos

  • Campo (1): Mostra o saldo todo contratado, disponível para ser transferido para um ou mais usuários do sistema.
  • Campo (2): É composto de uma caixa de seleção, através da qual é possível buscar um dos usuários do sistema para o qual se deseja fazer transferências de créditos de mensagens de texto.
  • Campo (3): Destinado a inserção da quantidade de mensagens a ser transferida para o usuário em questão.
  • Campo (4): Nesta caixa suspensa você escolhe o tipo de transferência desejado.
    • Se você escolher crédito, a quantidade escolhida de mensagens por você será creditada ao usuário escolhido e debitada do saldo total disponível.
    • Se você escolher débito, a quantidade escolhida de mensagens por você será debitada do usuário escolhido e creditada ao saldo total disponível.

Após feitos todas esses procedimentos para transferência de saldos para ou de usuários, basta clicar no botão Gravar para confirmar a transferência.
