Cadastrando tipos de Sugestão

Continuando com os procedimentos de implantação da Plataforma de Gestão Colaborativa Prefeitrura.com.vc, agora é a hora de criar as sugestões e conectá-las a um dos setores ativos para que a mesma seja tratada quando de uma demanda por um cidadão.

Nosso conselho é que nessa hora você que está implantando em seu município faça uma planilha com as sugestões escolhidas, divida-as por Departamentos e por setores para evitar duplo cadastramento ou falta de cadastramento de alguma delas.

Para tanto vá ao menu Tipo de sugestões e, apontando o mouse para o botão vermelho, no canto inferior direito, clique no botão verde com o sinal de “+”. A tela de criação de sugestões estará visível agora.

Para criar a sugestão você deve seguir os campos descritos no formulário. A primeira informação a ser cadastrada é o Departamento para o qual você irá destinar a solução do tipo de sugestão cvadastrado, para isso lcique na seta que está circulada em vermelho.

A próxima informação é o nome da Sugestão, nome que aparecerá para o cidadfão no aplicativo para dispopsitivos móveis ou no Assistyente Virtual pelço WhatsApp. Escreva o nome no campo chamado “descrição”.

Algumas cidades maiores, dividem o seu território em Prefeituras Regionais ou Sub prefeituras, que congregam certa área do município formada por um dado número de bairros.

Na sequência você fornece uma informação importante para a Plataforma Prefeitrura.com.vc, você diz se seu município tem ou não Prefeituras Regionais e portanto se aquela sugestão estará disponível para todos os bairros (Prefeitura Municipal) ou se estará disponível apenas para as regiões cobertas pelas Prefeituras Regionais.

A próxima informação que você vai inserir o setor para o qual essa sugestão será encaminhada. Para tanto clique na caixa de seleção e escolha um dos setores ativos.

O próximo passo é juntamente com o gestor de sua Unidade Gestora determinar qual é o prazo máximo de execução daquela sugestão, a partir do qual a ausência de cumprimento da mesma será considerado atraso.

Finalmente você informará se aquela sugestão irá ou não aparecer imediatamente no Aplicativo para Dispositivos Móveis ou no Assistente Virtual para WhatsApp dos cidadãos e para isso você escolherá entre os status ATIVO ou INATIVO.

Ao clicar no botão ENVIAR sua sugestão estará completamente criada.

Editando os parâmetros de um Tipo de sugestão

Lembre-se que a Plataforma Prefeitrura.com.vc é totalmente flexível. Cada tipo de sugestão pode ter seus parâmetros alterados quando, ou se necessário.

Para editar um tipo de sugestão qualquer clique no menu "Sugestões", para ver a tela abaixo:

Clicando no lápis a direita de cada sugestão, conforme exemplo assinalado com círculo vermelho, você tem acesso à tela de edição, onde cada uma das características originalmente cadastradas pode ser alterada.

Após alterar qualquer um dos itens do formulário, clique no botão ENVIAR e sua sugestão estará completamente editada e as informações alteradas serão salvas.
