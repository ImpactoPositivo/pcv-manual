Definição de Setor

Um setor dentro de um Departamento, que faz parte de uma Prefeitura, é uma subdivisão ou unidade organizacional criada para gerenciar e coordenar tarefas específicas dentro de uma área específica de atuação governamental. Cada Departamento, por sua vez, é responsável por uma área de interesse municipal, como educação, saúde, segurança, meio ambiente, entre outras. Esses setores, muitas vezes, são criados para promover uma divisão de trabalho eficaz e melhor gerenciamento das atividades dentro de uma secretaria.

Cadastrando um Setor

O cadastramento de setores no sistema Prefeitura.com.vc se dá pelo gestor de um Departamento administrativo, ou seja, uma Secretaria, Autarquia ou Empresa Terceirizada cadastrada no sistema com poderes para criar setores.

Para cadastrar um Setor, o gestor de 2º escalão deve clicar no ítem Setor, para abrir a tela de cadastramento conforme imagem abaixo:

Preencha completamente o formulário da tela de cadastramento informando primeiramente o Departamento para o qual você deseja vincular operacionalmente o Setor.

Continue o cadastramento com o nome do Setor no item "Descrição".

Escolha a opção "ATIVA" e clique no botão ENVIAR para criar o Setor.

Para atribuir a administração de cada setor criado, é necessário vincular o mesmo a um servidor cadastrado no sistema.
